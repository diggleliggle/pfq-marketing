How it works

The directory updates automatically.

Members do not manually update their listing. It reflects the live state of their IPP record.

On joining

A verified listing is created automatically when membership is confirmed. Name, tier, designation, and join date are populated from the IPP record.

On qualifying

Passing an IPP qualification updates the Calibrate profile and triggers an automatic tier upgrade prompt. The directory listing updates on confirmation.

On CPD activity

CPD hours logged in Calibrate update CPD currency status in real time. If CPD lapses, the designation shows as inactive until restored.
